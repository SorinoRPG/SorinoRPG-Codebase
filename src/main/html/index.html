F.A.Q.

Frequently Asked Questions

  • How do I add SorinoRPG to my server?

    You can add SorinoRPG to your server using this invite link

  • How do I change the prefix?

    You can change SorinoRPG's prefix with .setprefix [PREFIX]

  • SorinoRPG won't respond, what do I do?

    You should first ping the bot, to make sure you are using the command properly. If you are, you can report this using the bug reporting feature, or directly report it to staff.

  • How can I report Bugs?

    One way is to directly go to a developer and tell them about the bug. Another way is to enter .bug [THE BUG].

  • How do I stop SorinoRPG spamming level ups?

    You can always enter .setlevel OFF to completely turn level up off. If you want to redirect it to another channel, enter .setlevel #channel
